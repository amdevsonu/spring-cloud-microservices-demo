# Spring Cloud Microservices Demo Application

## Overview
A microservices-based backend application built using Spring Boot and Spring Cloud components,
demonstrating service discovery, centralized configuration, API gateway routing,
inter-service communication, fault tolerance, and observability.

## Architecture
- Service Registry (Eureka Server)
- Config Server (Spring Cloud Config)
- API Gateway (Spring Cloud Gateway)
- Admin Server (Spring Boot Admin)
- Distributed Tracing (Zipkin)
- Welcome Service
- Greet Service

## Technologies Used
- Java
- Spring Boot
- Spring Cloud (Eureka, Config Server, Gateway, OpenFeign)
- Resilience4j
- Zipkin
- Maven

## How to Run the Project
1. Start Config Server
2. Start Eureka Server
3. Start Admin Server
4. Start Zipkin
5. Start API Gateway
6. Start Welcome Service
7. Start Greet Service
